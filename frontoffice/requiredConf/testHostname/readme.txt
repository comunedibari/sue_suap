Dopo l'installazione di SUAP SUE se si verifica l'errore:

setNomeServer is null

quando si prova a invocare il back end, es. DOC_DINAMICI o WS_INDIRIZZO o WS_CATASTO. 

Bisogna lanciare il comando da terminale sulla macchina di installazione:

hostname

il cui risultato deve essere messo nel file hosts associato all'indirizzo 127.0.0.1
