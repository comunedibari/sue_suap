Configurazione Browser - Mozilla Firefox
Guida alla configurazione della firma digitale sul portale P.E.O.P.L.E.
Dopo aver configurato le politiche di sicurezza sul download e installazione degli ActiveX, per poter firmare con Mozilla Firefox è necessario seguire questa procedura:
  • installare sul PC client un componente aggiuntivo, scaricabile gratuitamente da questo sito. Per accedere direttamente alla pagina di download è possibile seguire questo link
  • modificare il file activex.js nella cartella <root_installazione_Firefox>\default\pref (e.g. c:\programmi\Mozilla Firefox\default\pref\activex.js) aggiungendo le seguenti righe

    pref("capability.policy.default.ClassID.CIDA996E48C-D3DC-4244-89F7-AFA33EC60679","AllAccess"); pref("capability.policy.default.ClassID.CIDF0A8EC0B-68F2-4819-B2CC-40AEBF2D6296","AllAccess");
